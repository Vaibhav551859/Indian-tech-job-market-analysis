# Indian Tech Job Market Analysis 2026

## Project Overview

This project analyzes 23,201 technology job postings across India to identify hiring trends, salary patterns, experience requirements, skill demand, and work arrangements.

The analysis was performed using Python, Pandas, and Power BI.

---

## Dataset

- Source: Kaggle
- Records: 23,201
- Features: 32 columns
- Categories:
  - Data Scientist
  - Data Analyst
  - Business Analyst
  - Machine Learning Engineer
  - Data Engineer
  - Python Developer

---

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Power BI
- GitHub

---

## Key Insights

- Mumbai is the largest hiring market with 3,276 openings.
- Data Scientist is the most in-demand role.
- Mid-level professionals (3–5 years) account for 44% of openings.
- Only 18% of jobs are fresher-friendly.
- Cloud & DevOps offers the highest average salary (19.02 LPA).
- On-site work represents over 80% of job postings.
